# 模板说明

本目录提供资料库统一模板，用于保证条目结构稳定、字段一致、便于长期扩写。

## 模板列表

- `era-template.md`：地质年代与时代综述页
- `clade-template.md`：类群与演化支系页
- `species-template.md`：代表物种与关键案例卡片
- `source-tool-template.md`：来源工具、矩阵、清单与规则页

## 使用原则

- 新建条目时先复制最接近的模板，再删去不适用字段。
- 资料库 2.0 起，优先先填写 frontmatter，再写正文。
- 不要随意修改一级结构；若确有必要，优先新增小节而不是重写整套模板。
- 每个条目末尾都应包含“核心来源”和“争议与说明”。
- 若某字段当前缺失，可写“待补”，但应尽快用来源补齐。
