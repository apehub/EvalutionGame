# 素材资产层

`07-assets` 用来承接资料库后续的图片、图示、模型、时间轴素材和相关版权说明。

它的目标是让素材与正文解耦：正文负责解释和叙述，素材层负责管理可复用资源及其来源、授权和挂接关系。

## 计划承接的内容

- 条目封面图与缩略图
- 示意图、结构图、流程图
- 时间轴视觉素材
- 3D 模型、骨架重建和贴图参考
- 素材来源、版权、许可和挂接说明

## 当前入口

- `asset-schema.md`：素材字段规范
- `asset-registry.md`：素材注册表原型

## 使用原则

- 素材通过 `asset_id` 与正文条目关联。
- 同一素材可服务多个条目，不应只在单篇正文里孤立描述。
- 所有对外可用素材都应记录来源和许可边界。
