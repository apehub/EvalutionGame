# 前寒武纪关键节点时间表

本时间表用于辅助 `origin-of-life.md`、`precambrian-overview.md`、`early-life.md` 和 `great-oxidation-event.md` 等条目的写作与核查。它的目标不是给出最激进年代，而是整理资料库当前阶段更适合采用的“稳妥时间口径”。

## 使用原则

- 优先写主流、较稳妥的时间范围，不把争议最大的超早说法直接写进正文主干。
- 每个节点都要区分“环境条件出现”“生命证据出现”“复杂性提升”三类不同问题。
- 时间表是写作工具，不替代具体条目中的争议说明。

## 关键节点

| 节点 | 建议时间口径 | 证据层级倾向 | 主要用途 | 写作提醒 |
| --- | --- | --- | --- | --- |
| 地球形成后较早稳定环境 | 冥古宙晚期前后可出现液态水与较稳定地壳条件 | `A` 到 `B` | 作为生命起源讨论的环境上限 | 不要直接写成生命已经出现 |
| 最早较稳妥生命线索 | 太古宙早期范围内表述更稳妥 | `B` | 作为“生命已存在”的保守起点 | 避免直接采用最激进超早记录 |
| 微生物生态系统明确可见 | 太古宙中期以后更容易写得稳妥 | `A` 到 `B` | 用于支撑叠层石、微生物席等材料层说明 | 适合与叠层石卡片联动 |
| 产氧光合作用的重要扩张 | 太古宙后期到元古宙早期之间可作为长期过程写 | `B` | 作为大氧化事件的前置背景 | 不要压缩成单一瞬时点 |
| 大氧化事件主窗口 | 元古宙早期约 `2.4 Ga` 到 `2.0 Ga` | `A` 到 `B` | 支撑环境转折条目 | 强调长期过程而非单点爆发 |
| 真核生物进入更清晰记录 | 元古宙阶段内写成逐步进入记录更稳妥 | `B` | 支撑复杂细胞层级出现 | 避免写成单一明确瞬间 |
| 多细胞化前奏 | 元古宙中晚期作为渐进背景更稳妥 | `B` 到 `C` | 支撑复杂生命前奏 | 不要直接写成现代动物意义上的多细胞生态成熟 |
| 埃迪卡拉型复杂生物出现 | 前寒武纪末期可作为复杂多细胞生命的重要代表 | `A` 到 `B` | 作为寒武纪前夕的关键节点 | 应与寒武纪大爆发分开写 |
