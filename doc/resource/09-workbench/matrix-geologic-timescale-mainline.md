# 完整地质时间骨架主线图

本表用于把“完整地质时间骨架面”拆成可逐步推进的几条线，而不是直接零散补纪级页面。

## 使用原则

- 先补能把整条时间主轴串起来的线，再补单个纪级点。
- 每补一条线，都应能让 `01-timeline/` 的时间连续性更强，而不是只增加孤立页面。
- 时间骨架页优先回答“这一段在全局生命史里承担什么位置”，再补更细节的支系和事件互链。

## 当前线级拆分

### 线一：古生代连续骨架线

- 当前状态：最缺，优先级最高。
- 现有节点：`cambrian.md`、`devonian.md`、`permian-end-crisis.md`
- 主要缺口：`ordovician.md`、`silurian.md`、`carboniferous.md`，以及把 `permian-end-crisis.md` 旁边补出更完整的二叠纪时代页。
- 这条线的作用：把“寒武纪 -> 奥陶纪 -> 志留纪 -> 泥盆纪 -> 石炭纪 -> 二叠纪”串成连续古生代主轴。
- 页面职责入口：`matrix-paleozoic-mainline-page-roles.md`

### 线二：中生代纪级展开线

- 当前状态：只有代级总览，缺内部纪级节点。
- 现有节点：`mesozoic-overview.md`
- 主要缺口：`triassic.md`、`jurassic.md`、`cretaceous.md`
- 这条线的作用：把“中生代总览”拆开成灾后重建、恐龙扩张、白垩纪复杂化与终局三段。

### 线三：新生代纪级展开线

- 当前状态：只有代级总览，缺内部阶段骨架。
- 现有节点：`cenozoic-overview.md`
- 主要缺口：`paleogene.md`、`neogene.md`、`quaternary.md`
- 这条线的作用：把“现代生态如何形成”拆成灾后辐射、现代生态重塑、冰期与人类阶段三段。

### 线四：代级与纪级互链修正线

- 当前状态：需要和前面三条线并行推进。
- 主要任务：让 `precambrian-overview.md`、古生代纪级页、中生代纪级页、新生代纪级页形成清晰上下级阅读顺序。
- 这条线的作用：避免时间页继续各写各的，保证“上一时代 / 下一时代 / 本段核心事件”表达统一。

## 推荐推进顺序

1. 古生代连续骨架线
2. 中生代纪级展开线
3. 新生代纪级展开线
4. 代级与纪级互链修正线

## 每条线的最小完成标准

- 至少补齐该线最关键的缺失时间页名称与顺序。
- 至少明确每一页主要回答什么，而不是只留下年代名称。
- 至少给已有总览页补回清晰的上下位阅读路径。
