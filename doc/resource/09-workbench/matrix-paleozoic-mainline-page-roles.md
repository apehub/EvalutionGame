# 古生代连续骨架线页面职责表

本表用于把“古生代连续骨架线”拆成更具体的页面职责，避免后续补 `ordovician.md`、`silurian.md`、`carboniferous.md`、`permian.md` 时与现有时间页或事件页重复。

## 使用原则

- 每一页先回答“这一纪在整条古生代主线中承担什么位置”。
- 纪级页写时代主干，事件页写转折机制，支系页写长期演化主线，物种卡写代表证据。
- 若某个内容更适合成为事件页，不应全部压进纪级页里。

## 现有锚点页职责

### `cambrian.md`

- 主要职责：显生宙起点、海洋动物门级多样化、寒武纪大爆发入口。
- 不应过度承担的内容：奥陶纪海洋多样化延续、后续古生代长期生态升级。

### `devonian.md`

- 主要职责：鱼类辐射、植物扩张、脊椎动物登陆三线交汇。
- 不应过度承担的内容：志留纪早期陆地生态奠基、石炭纪森林生态成熟、二叠纪危机前总览。

### `permian-end-crisis.md`

- 主要职责：古生代末危机专题、生态崩塌与中生代重建前夜。
- 不应过度承担的内容：整个二叠纪时代总览；这部分应由未来 `permian.md` 承担。

## 待建页职责

### `ordovician.md`

- 在线中的位置：寒武纪之后的第一段海洋扩张期。
- 主要回答：
  - 寒武纪之后海洋多样化如何继续扩展
  - 哪些无脊椎主线在这一纪显著增强
  - 奥陶纪末危机为何会成为下一阶段的重要转折前奏
- 推荐互链：
  - `cambrian.md`
  - `invertebrates.md`
  - `mass-extinctions-overview.md`
- 不应写成：单纯“寒武纪大爆发续集”

### `silurian.md`

- 在线中的位置：从海洋主线向陆地生态初步建立过渡的桥页。
- 主要回答：
  - 奥陶纪末之后生态如何恢复
  - 早期陆地植物和陆生生态雏形为何在这一纪值得单独提出
  - 泥盆纪植物扩张与登陆线为何要从这里预热
- 推荐互链：
  - `ordovician.md`
  - `plants.md`
  - `plant-landfall.md`
  - `devonian.md`
- 不应写成：被泥盆纪完全吞掉的前奏页

### `carboniferous.md`

- 在线中的位置：陆地森林生态与羊膜动物起步的关键扩展段。
- 主要回答：
  - 泥盆纪之后陆地生态系统如何进一步立体化
  - 森林、湿地、高氧环境等叙事为什么在这一纪特别重要
  - 羊膜动物起步与后续陆地脊椎动物扩展如何在这里取得新阶段
- 推荐互链：
  - `devonian.md`
  - `plants.md`
  - `amphibians.md`
  - 后续羊膜动物相关条目
- 不应写成：仅仅是“泥盆纪之后的陆地延长线”

### `permian.md`

- 在线中的位置：古生代主线的终章总览页。
- 主要回答：
  - 二叠纪在盘古大陆背景下的环境与生态结构
  - 哪些古生代主导格局在这一纪达到高峰或开始失稳
  - 为什么二叠纪必须和 `permian-end-crisis.md` 分成“时代总览页 + 危机专题页”
- 推荐互链：
  - `carboniferous.md`
  - `permian-end-crisis.md`
  - `mass-extinctions-overview.md`
  - `reptiles.md`
- 不应写成：把危机专题页内容整块复制过来
