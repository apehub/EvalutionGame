# 地质时间骨架待建顺序表

本表用于回答：如果现在开始补“完整地质时间骨架面”，应先补哪些时间页，后补哪些时间页。

## 使用原则

- 先补能最大幅度消除时间断层的页面。
- 同一轮尽量补相邻时间页，而不是跨得太散。
- 若某个阶段已有人气专题页或关键事件页，优先把它所在的时间骨架补完整。

## 第一批优先待建

### 古生代连续骨架补齐

1. `ordovician.md`
2. `silurian.md`
3. `carboniferous.md`
4. `permian.md`

说明：

- 这 4 页补上后，古生代将首次接近连续主轴。
- 其中 `permian.md` 应和现有 `permian-end-crisis.md` 分工明确：前者是时代总览，后者是危机专题节点。
