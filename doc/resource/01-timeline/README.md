# 时间主轴

本目录用于按地质年代整理生命史主线，优先建立“能支撑全局叙事”的骨架页，再逐步细化。

## 第一阶段优先条目

### 起点与前寒武纪

- `origin-of-life.md`：生命起源与最早生命证据
- `precambrian-overview.md`：前寒武纪总览

### 古生代关键节点

- `cambrian.md`：寒武纪与寒武纪大爆发
- `devonian.md`：泥盆纪与鱼类时代、四足动物起源
- `permian-end-crisis.md`：二叠纪末危机与生态系统重置

### 中生代与新生代

- `mesozoic-overview.md`：中生代总览
- `cenozoic-overview.md`：新生代总览

## 编写顺序建议

1. `origin-of-life.md`
2. `precambrian-overview.md`
3. `cambrian.md`
4. `devonian.md`
5. `permian-end-crisis.md`
6. `mesozoic-overview.md`
7. `cenozoic-overview.md`
