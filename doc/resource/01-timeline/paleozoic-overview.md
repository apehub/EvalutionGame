# 古生代总览（Paleozoic Overview）

## 基本信息

- 时间范围：约 `538.8 Ma` 到 `251.9 Ma`
- 所属层级：代
- 上一时代：前寒武纪
- 下一时代：中生代
- 内部纪级：寒武纪、奥陶纪、志留纪、泥盆纪、石炭纪、二叠纪
- 主要标签：`marine` `terrestrial` `cambrian` `devonian` `permian` `amniotic-egg`

## 时代概述

古生代是显生宙前半段的第一整代，也是生命从“海洋中建立复杂动物生态系统”逐步推进到“陆地生态系统成形”的关键大阶段。它既包含寒武纪和奥陶纪这样以海洋扩张为主的阶段，也包含泥盆纪到石炭纪这样把植物、四足动物和羊膜动物拉上陆地舞台的阶段，最终以二叠纪末危机结束。

如果只把古生代理解成“三叶虫和早期鱼类的时代”，会漏掉它真正的结构意义。古生代最重要的主题，是动物生态系统在海洋中成形、陆地生物圈逐渐复杂化，以及脊椎动物从水域主线转向陆地主线的长期铺垫。到了晚古生代，森林生态、羊膜动物和更复杂的陆地食物网已经为中生代新格局准备了底盘。

从资料库结构看，这一页不替代具体纪级页面，而是承担总代入口职责：它需要把寒武纪到二叠纪串成连续阅读顺序，并把相关支系页、事件页和专题页接入统一主线。

## 地球环境

### 气候与大气

- 古生代内部气候差异很大，不能用单一“温暖”或“寒冷”概括。
- 大气氧含量、生物泵和陆地植被扩张在这一代内部都有显著变化。

### 海陆格局与地质背景

- 早期仍以广阔浅海生态系统为主。
- 晚古生代大陆拼合趋势增强，陆地环境和内陆生态作用上升。

### 主要生态系统

- 寒武纪到奥陶纪以海洋生态系统复杂化为主。
- 志留纪到泥盆纪陆地植物和近岸生态位的重要性迅速上升。
- 石炭纪到二叠纪森林、沼泽、干旱内陆和更复杂的陆地脊椎动物系统并存。

## 古生代内部主线

### 早古生代：海洋扩张

- 寒武纪建立显生宙动物生态系统的可见框架。
- 奥陶纪推动海洋多样化继续扩展。
- 志留纪开始出现更稳固的陆生生态前奏。

### 中晚古生代：登陆与陆地复杂化

- 泥盆纪是鱼类辐射、植物扩张和四足起源交汇的关键期。
- 石炭纪强化森林生态和陆地食物网，并让羊膜动物起步。
- 二叠纪把晚古生代格局推向更大尺度整合，并以末期危机完成一次系统重置。

## 关键演化转折

- 寒武纪大爆发使动物生态系统进入高度可见阶段。
- 植物登陆逐步改变土壤、风化和陆地生态结构。
- 脊椎动物登陆把演化主线从海洋推进到海陆双线。
- 羊膜动物起步为后续爬行动物和哺乳动物主线奠定基础。
- 二叠纪末危机终结古生代主导格局，并打开中生代重建阶段。
