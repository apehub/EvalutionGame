# 时间索引

本表用于把时间主轴、关键事件、代表卡片和支系页按时间窗口做统一索引。它服务于后续 HTML timeline、筛选器和排序逻辑。

## 使用原则

- 时间统一优先换算为 `Ma`。
- 若时间跨度很长，可用大范围表示，再由正文细化具体阶段。
- 时间索引不是正文，不负责展开完整解释。
