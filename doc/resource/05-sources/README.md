## 使用顺序

1. 先用 `web-authoritative.md` 找到条目的主要在线入口。
2. 再用 `books-core.md` 补足学科框架与专题深挖书。
3. 最后按 `citation-rules.md` 给条目安排来源与证据等级。
4. 若条目涉及模型分歧、分类变化或通俗误解，再按 `controversy-recording-rules.md` 统一写“争议与说明”。
5. 若一个来源会反复使用，先登记到 `source-registry.md`；若需要强化条目内部写法，则按 `evidence-layer-template.md` 分层整理。
6. 若要继续做深挖考据，可直接按四份 `checklist-*.md` 清单逐篇核查。
7. 若要补强鱼类登陆与四足动物起源相关条目，可直接参考 `matrix-vertebrate-landfall-features.md`。
8. 若要区分关键过渡节点之间的写法差异，可直接参考 `matrix-tiktaalik-vs-ichthyostega.md`。
9. 若要快速把握过渡序列整体结构，可直接参考 `matrix-landfall-sequence-core.md`。
10. 若要补强提塔利克鱼之前的背景阶段，可直接参考 `matrix-landfall-earlier-nodes.md`。
11. 若要继续细化肉鳍鱼相关近缘层，可直接参考 `matrix-sarcopterygian-nearshore-nodes.md`。
12. 若要防止正文写得过满或过度戏剧化，可直接参考 `matrix-landfall-writing-phrases.md`。
13. 若要补强前寒武纪主轴的时间口径，可直接参考 `matrix-precambrian-key-nodes.md`。
14. 若要区分前寒武纪不同证据能支持什么，可直接参考 `matrix-precambrian-evidence-types.md`。
15. 若要避免前寒武纪条目写得过度戏剧化，可直接参考 `matrix-precambrian-writing-phrases.md`。
16. 若要专门防止叠层石条目把结构型证据写成生物定论，可直接参考 `matrix-stromatolite-misreadings.md`。
17. 若要区分大氧化事件中哪些证据属于环境转折主干、哪些属于解释层，可直接参考 `matrix-goe-evidence-roles.md`。
18. 若要区分生命起源页中哪些证据属于环境上限、生命线索或模型解释层，可直接参考 `matrix-origin-evidence-roles.md`。
19. 若要区分早期生命页中哪些证据属于主导背景、环境改造或复杂性前提，可直接参考 `matrix-early-life-evidence-roles.md`。
20. 若要快速把握前寒武纪五个核心页面的职责和阅读顺序，可直接参考 `matrix-precambrian-mainline-reading-order.md`。
21. 若要防止前寒武纪主线五个核心页面的职责写串，可直接参考 `matrix-precambrian-mainline-misreadings.md`。
22. 若要避免早期生命页把微生物主导背景、环境改造和复杂性前提写得过满，可直接参考 `matrix-early-life-writing-phrases.md`。
23. 若要避免大氧化事件页把长期转折写成瞬时跳变或单因果结论，可直接参考 `matrix-goe-writing-phrases.md`。
24. 若要避免生命起源页把环境上限、生命线索和模型解释写串，可直接参考 `matrix-origin-writing-phrases.md`。
25. 若要快速查某个前寒武纪核心页面该配哪组工具，可直接参考 `matrix-precambrian-core-pages-toolmap.md`。
26. 若要先把自然语言问题句式对应到前寒武纪主题入口，可直接参考 `matrix-precambrian-question-phrases.md`。
27. 若要从“生命起点”“微生物主导”“环境氧化转折”等主题直接反查页面与工具，可直接参考 `matrix-precambrian-topic-entrypoints.md`。
28. 若要在写叠层石卡前先判断哪些材料能进正文、哪些应降级为说明，可直接参考 `checklist-stromatolite-materials.md`。
