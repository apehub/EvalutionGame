# 叠层石材料层排查清单

本清单用于专门核查 `stromatolite-microbial-mats.md` 这类结构型代表卡的材料层写法。它的目标不是补充更多理论，而是帮助在下笔前快速判断：哪些内容可以稳妥进入正文，哪些需要降级为说明项，哪些应继续核对后再写。

## 一、必核项目

### 结构身份

- 是否明确说明叠层石不是单一物种，而是结构型条目
- 是否先从层状、穹隆状、柱状等可见结构写起，而不是一开始就跳到生物结论

### 材料层拆分

- 是否把材料区分为结构证据、显微与群落线索、地球化学与环境线索三层
- 是否避免把不同类型证据混写成同一确定性等级

### 环境背景

- 是否说明典型沉积环境，如浅海、潮坪、受限海湾等
- 若使用现代类比，是否明确写出“只能作为机制参考，不等同于前寒武纪原场景”

### 争议边界

- 是否区分“较广泛接受的典型案例”和“最古老记录争议案例”
- 是否明确说明外观相似构造不自动等于生物成因

## 二、可以稳妥进入正文的材料

- 层状、穹隆状或柱状沉积构造等结构描述
- 微生物席长期改造沉积表面的背景性说明
- 浅海和沉积环境中的长期稳定生物活动窗口
- “它是早期微生物生态系统的重要结构型证据之一”这一层级的判断

## 三、应降级为说明或争议项的内容

- “它就是最早生命本身”的断言
- “某个极早年代记录已完全无争议”的断言
- 仅凭单类材料就写成“生物成因已最终定案”
- 仅凭叠层石就写成“最早光合作用或全球氧化问题已经解决”

## 四、推荐来源组合

- Knoll
- UCMP
- Smithsonian Ocean
- Paleobiology Database
- 若继续深挖材料层，应补充沉积学或地球化学专题来源

## 五、核查提醒

- 先写“看到了什么结构”，再写“为什么更像生物活动”。
- 若一句话同时包含结构描述、年代定论和生理推断，优先拆开。
- 若某项判断强依赖争议案例，应先降级为说明项。
- 若需要处理常见误写，优先联动 `matrix-stromatolite-misreadings.md`。
- 若需要判断证据支持边界，优先联动 `matrix-precambrian-evidence-types.md`。
