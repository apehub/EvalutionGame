# 前寒武纪常见问题句式 vs 对应入口主题

本表用于把自然语言里的常见提问方式，快速对应到前寒武纪资料库已经建立的主题入口。它的目标不是直接替代正文，而是帮助先判断“这句话本质上在问哪类问题”，再跳转到 `matrix-precambrian-topic-entrypoints.md` 中对应的主题行。

## 使用原则

- 先按左列找到与你当前问题最接近的句式，再看它更适合落在哪个入口主题。
- 若一句话同时碰到多个主题，优先按“核心想回答什么”归类，再把其他问题拆到下一步处理。
- 本表处理的是“问题分流”，不是“证据定论”；进入主题后仍应继续查对应页面和工具。

## 推荐使用方式

### 1. 先有一句问题，但不知道该进哪类主题

- 先在本表左列找最接近的自然语言句式。
- 再按右列跳到 `matrix-precambrian-topic-entrypoints.md` 对应主题行。

### 2. 一句话里混了两个层次

- 先看它更像在问起点、背景、转折、复杂性还是材料。
- 若仍混杂，优先拆成两句分别进入不同主题。

### 3. 进入主题后还是不确定写法

- 继续顺着 `matrix-precambrian-topic-entrypoints.md` 的“下一步建议看的工具”往下查。
- 若担心口语问题句本身带偏写法，再回看 `matrix-precambrian-writing-phrases.md`。
