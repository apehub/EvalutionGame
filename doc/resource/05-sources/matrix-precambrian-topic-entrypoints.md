# 前寒武纪问题主题反查表

本表用于把“我现在要写什么主题”直接反查到前寒武纪资料库中最适合先进入的页面和工具。它不是主线阅读顺序表，也不是页面工具总表，而是一个按内容主题组织的反向入口层，帮助在不先记住页面职责的情况下快速起步。

## 使用原则

- 先确认你当前卡住的是哪个主题，而不是先猜应该打开哪一页。
- 每个主题先进入一个最合适的页面，再按“下一步建议看的工具”补证据层或写法防护层。
- 如果写着写着发现主题边界混了，优先回看最后一列中的总控表。

## 推荐使用方式

### 1. 已经知道要写什么主题，但不知道先看哪页

- 直接按上表第一列定位主题。
- 先进入第二列页面，再决定是否需要补证据工具或写法防护工具。

### 0. 还停留在自然语言问题句式

- 若你当前只有一句口头问题，还没判断它属于哪类主题，先看 `matrix-precambrian-question-phrases.md`。
- 再从那张表跳回本表对应主题行。

### 2. 知道主题，但不知道证据够不够

- 优先进入对应主题行中的“下一步建议看的工具”。
- 若该工具仍不足以判断，再回看 `matrix-precambrian-evidence-types.md`。

### 3. 写着写着感觉主题串位了

- 先回看该主题行中的“必要时回看的总控表”。
- 若仍拿不准，再返回 `precambrian-overview.md` 重新确认主线位置。
