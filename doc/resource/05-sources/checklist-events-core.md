# 关键事件证据对照清单

本清单用于核查事件页中“事件确实发生了”与“事件为何这样发生”的两层内容，避免把成因模型写成确定事实。

## 一、`great-oxidation-event.md`

### 必核项目

- 时间窗口是否按主流范围表达
- 条带状铁建造、同位素、红层等证据是否被区分为不同证据类型
- 是否明确写出“长期过程”而非“单点爆发”

### 推荐来源组合

- Knoll：前寒武纪背景
- UCMP：教学型综述
- USGS：方法与地层证据说明

### 核查提醒

- 氧含量上升不等于全球环境同步完全富氧

## 二、`cambrian-explosion.md`

### 必核项目

- “爆发”是否被写成相对快速而非瞬时
- 环境、发育、生态、保存偏差四类解释是否有清楚边界
- 是否避免把所有门类都写成同一时刻出现

### 推荐来源组合

- Erwin / Valentine：专题核心
- UCMP：综述
- Clarkson：无脊椎动物背景

### 核查提醒

- 优先保留多因素解释框架

## 三、`vertebrate-landfall.md`

### 必核项目

- 是否区分“过渡序列”与“单一祖先链条”
- 是否把环境背景、骨骼变化和生态位变化分开写
- 是否避免把登陆写成一次性脱离水环境

### 推荐来源组合

- Benton：脊椎动物总览
- Clack：四足动物起源专题
- PBDB：产地与时间辅助核查

### 核查提醒

- “近缘代表”比“直接祖先”通常更稳妥

## 四、`mass-extinctions-overview.md`

### 必核项目

- 五次大灭绝是否作为“教学总表”而不是精细专题替代页
- 是否明确区分“灭绝规模”与“后续生态重排”
- 是否避免把五次危机写成同等规模与同等机制

### 推荐来源组合

- Benton / Harper：总体框架
- UCMP：生命史综述
- PBDB：多样性与分布辅助

### 核查提醒

- 总览页的目标是搭框架，不是把每次危机的争论一次写完
