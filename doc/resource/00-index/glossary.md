# 术语表

## 地史与演化基础

- 地质年代（geologic time scale）：用于划分地球历史的标准时间体系。
- 演化支系（clade）：包含某共同祖先及其全部后裔的自然类群。
- 系统发育（phylogeny）：生物类群之间祖先与后代关系的历史。
- 适应辐射（adaptive radiation）：一个支系在较短时期内迅速分化并占据多个生态位。
- 大灭绝（mass extinction）：全球范围内大量类群在相对短时间内消失的事件。

## 生命起源与早期生命

- 原核生物（prokaryotes）：没有细胞核的生物，包括细菌和古菌。
- 真核生物（eukaryotes）：具有细胞核和复杂细胞器的生物。
- 多细胞化（multicellularity）：多个细胞形成分工协作个体的演化过程。
- 大氧化事件（Great Oxidation Event）：元古宙早期大气氧含量显著升高的关键转折。

## 物种与类群条目常用词

- 学名（scientific name）：符合生物命名规则的正式名称。
- 拉丁学名：中文语境中常用于指代生物学正式学名。
- 模式标本（type specimen）：用于定义某一分类单元名称的标准标本。
- 过渡型（transitional form）：在关键演化节点上显示祖先和后代特征组合的类型。
