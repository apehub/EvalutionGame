# 全库主题面地图

本页用于把资料库从“按条目类型看”切换到“按主题面看”。

这里的“面”，指能够承接多条主线和多个条目类型的一级主题板块。一个面通常不只包含单篇正文，而应至少能容纳：

- 时间主轴页
- 支系页
- 事件页
- 代表卡
- 对应的工作台工具

## 使用原则

- 先看一个主题面是否已经成型，再决定是否继续往该面里补线和点。
- 不把“有单篇条目”误判成“已经成面”。
- 优先识别哪些面已经有时间、支系、事件、案例四类支撑中的两类以上。

- 当前重点：寒武纪主轴、寒武纪大爆发、早期动物代表。

### B 级：已有雏形但明显偏薄

#### 植物登陆 / 植物主线

- 状态：已有入口，但未成完整面。
- 当前已有：`../02-clades/plants.md`、`../03-events/plant-landfall.md`

- 状态：已有雏形，缺主干页。
- 当前已有：`../01-timeline/mesozoic-overview.md`、`../02-clades/reptiles.md`、`../02-clades/birds.md`、`../04-species-cards/archaeopteryx-lithographica.md`

#### 哺乳类主干

- 状态：已有总览层，缺中间分段。

- 状态：有总览，有代表危机，但还不是系列面。
- 当前已有：`../03-events/mass-extinctions-overview.md`、`../01-timeline/permian-end-crisis.md`

### C 级：几乎空白或尚未成面

#### 完整地质时间骨架

- 当前状态：断续节点，不是连续主轴。
- 主要缺口：奥陶纪、志留纪、石炭纪、二叠纪总览、三叠纪、侏罗纪、白垩纪和新生代内部阶段页。

#### 恐龙主干

- 当前状态：被分散在爬行动物、鸟类和始祖鸟之间。
- 主要缺口：恐龙面本体、主龙类分化线、兽脚类到鸟类线。

#### 海洋无脊椎细分主干

- 当前状态：仅有总览与寒武纪代表卡。
- 主要缺口：节肢动物、软体动物、棘皮动物、头足类等独立面或强线。

#### 人类演化

- 当前状态：几乎空白。
- 主要缺口：灵长类、人科、人属、古人类阶段主线与代表节点。

## 建议阅读顺序

1. 先看本页，判断当前要进入哪个主题面。
2. 再看 `index-map.md`，确认该面已有哪些条目。
3. 若该面已有工作台工具，再转到 `../09-workbench/` 中对应矩阵或清单。
4. 若该面仍未成型，优先回到 `surface-backlog.md` 判断是否先补面，而不是直接补零散条目。
