# 总索引

`00-index` 用来维护整套资料库的全局规则与入口索引。

## 文件列表

- `naming-rules.md`：目录、文件与条目命名规则
- `tag-system.md`：标签体系与交叉索引方法
- `geologic-timescale.md`：地质年代主骨架
- `glossary.md`：术语表与常用中英文对照
- `index-map.md`：全库已建/待建条目总表与互链检查入口

## 建议检索路径

- 想确认一个条目该放在哪个目录：先看 `naming-rules.md`
- 想知道一个条目应该挂哪些标签：先看 `tag-system.md`
- 想统一时代边界和时间书写：先看 `geologic-timescale.md`
- 想统一术语译名和书写方式：先看 `glossary.md`
- 想快速查看全库当前进度与条目互链：先看 `index-map.md`
