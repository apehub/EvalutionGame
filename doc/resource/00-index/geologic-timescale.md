# 地质年代主骨架

本文件只提供资料库用的主骨架，详细条目在 `01-timeline/` 内逐步扩写。年代边界以国际地层委员会（ICS）现行图表为准。

## 生命史主线

### 前寒武纪

- 冥古宙（Hadean）：地球形成与生命起源前条件
- 太古宙（Archean）：最早生命证据、原核生物主导
- 元古宙（Proterozoic）：大氧化事件、真核生物、多细胞生命前奏

### 古生代

- 寒武纪（Cambrian）：寒武纪大爆发、海洋动物门级多样化
- 奥陶纪（Ordovician）：海洋多样化扩张
- 志留纪（Silurian）：早期陆生生态初步建立
- 泥盆纪（Devonian）：鱼类辐射、植物扩张、四足动物起源
- 石炭纪（Carboniferous）：森林生态、羊膜动物起步
- 二叠纪（Permian）：盘古大陆背景与二叠纪末大灭绝

### 中生代

- 三叠纪（Triassic）：灭绝后重建、早期恐龙与哺乳形类
- 侏罗纪（Jurassic）：恐龙扩张、鸟类起源
- 白垩纪（Cretaceous）：被子植物兴起、白垩纪末灭绝

### 新生代

- 古近纪（Paleogene）：哺乳类与鸟类辐射重组
- 新近纪（Neogene）：现代生态系统成形、草原扩张
- 第四纪（Quaternary）：冰期循环与现代生物群

## 资料库使用说明

- 时间主轴条目先按“纪”级别建立，必要时再细化到“世”。
- 事件页可跨越多个时代，但必须在开头注明主要发生时间段。
- 类群页与物种卡片应明确“首次出现时间”和“主要辐射时期”。
