# 面级待建面板

本页用于追踪“哪些主题面已经成型，哪些还只是入口，哪些还没立起来”。

它不同于 `index-map.md`：

- `index-map.md` 主要追踪条目级已建 / 待建
- 本页主要追踪面级已成型 / 偏薄 / 待建立

## 使用方法

- 当准备继续扩库时，先看本页，再决定下一轮是补“面”、补“线”，还是补“点”。
- 若一个主题面还没有达到最低成面标准，不建议直接在其中零散追加太多点状条目。

## 判断标准

一个主题面若想从“待建立”升级到“偏薄”或“已成型”，至少应满足以下两项：

- 已有 1 个能承担面级入口作用的正文页
- 已有 1 条较明确的主线或关键事件线
- 已有 1 个代表案例卡或稳定对照工具
- 已有可帮助检索或写作的 workbench 辅助文件
