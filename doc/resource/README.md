# 地球生命演化资料库

这是一套面向“从生命起源到现代生物”的长期资料档案骨架，目标是为后续的研究、写作与游戏前期考据提供稳定的知识底座。

## 目录总览

- `00-index/`：总索引、术语表、地质年代对照、标签说明、命名规范
- `01-timeline/`：按地质年代整理的时间主轴
- `02-clades/`：按演化支系整理的类群主库
- `03-events/`：关键演化事件、环境转折与大灭绝
- `04-species-cards/`：代表物种与关键过渡类型卡片
- `05-sources/`：权威网络来源、核心书目、引文与证据规范
- `templates/`：时代页、类群页、物种页统一模板

## 使用原则

1. 先读 `00-index/README.md`，理解标签、命名与引用规范。
2. 时间问题优先查 `01-timeline/`，类群问题优先查 `02-clades/`。
3. 关键节点与机制问题优先查 `03-events/`。
4. 具体代表生物、过渡型与案例优先查 `04-species-cards/`。
5. 每个条目都应在文末列出核心来源，并标明争议点与证据等级。
