# 关键事件

本目录用于整理跨越时代与多个支系的关键演化事件、环境转折与大灭绝。

## 第一阶段优先条目

- `great-oxidation-event.md`：大氧化事件
- `cambrian-explosion.md`：寒武纪大爆发
- `plant-landfall.md`：植物登陆
- `vertebrate-landfall.md`：脊椎动物登陆
- `mass-extinctions-overview.md`：五次大灭绝总览
- `mammal-radiation.md`：哺乳类辐射

## 使用建议

- 每个事件页开头必须标注主要发生时间段。
- 事件页应说明“前因、过程、后果、受影响支系、证据与争议”。
- 大灭绝总览页完成后，可再拆出单独事件页，如二叠纪末与白垩纪末。
