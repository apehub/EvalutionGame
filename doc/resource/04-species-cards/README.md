# 代表物种卡片

本目录用于沉淀最细颗粒度的关键案例。第一阶段不追求数量，而优先选择“对理解演化节点最有价值”的代表物种或代表化石类型。

## 第一阶段建议优先卡片

### 早期生命与寒武纪

- `stromatolite-microbial-mats.md`：叠层石微生物群
- `anomalocaris-canadensis.md`：奇虾
- `trilobita-overview.md`：三叶虫总览型卡片

### 鱼类到四足动物

- `tiktaalik-roseae.md`：提塔利克鱼
- `ichthyostega.md`：鱼石螈

### 中生代到新生代

- `archaeopteryx-lithographica.md`：始祖鸟
- `morganucodon.md`：早期哺乳形类代表

## 选择标准

- 处在关键过渡节点
- 有较明确化石证据
- 能帮助理解大支系演化方向
- 适合后续扩展成游戏中的“标志性阶段生物”
