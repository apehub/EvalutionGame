# 真核生物（Eukaryotes）

## 基本信息

- 分类层级：主干桥梁页
- 系统位置：连接早期生命与复杂生命支线的关键总览页
- 起源时间：元古宙前后已有明确前奏
- 主要辐射时期：元古宙后期至今
- 是否现生：是
- 主要标签：`eukaryotes` `proterozoic` `multicellularity`

## 类群概述

真核生物是资料库主干中不可缺少的一座桥。没有这一页，前寒武纪主线会很容易从“早期生命”直接跳到植物、动物或寒武纪，从而漏掉复杂细胞结构出现这一关键层级。它的意义不只在于“细胞更复杂”，而在于后续多细胞化、大型化、组织分工和复杂生态关系，几乎都建立在真核细胞层级之上。

这页的职责不是细拆所有真核支系，而是回答三个问题：真核生物为什么是生命史中的结构升级、它与早期生命有何连续性、以及它为什么会成为植物和动物大支的共同前提。它是 `early-life.md` 与 `plants.md` / `animals.md` 之间的主干桥页。

## 关键特征

### 共有特征

- 拥有更复杂的细胞内部结构和分隔。
- 与后续多细胞化和功能分工关系紧密。

### 关键衍征

- 细胞器分工与复杂细胞结构整合。
- 更强的体积扩展和功能分化潜力。

### 生理与生态特征

- 既包含单细胞支线，也为后续多细胞大支系提供前提。
- 在海洋和后续陆地生态系统中都具有深远影响。

## 演化历史

### 起源背景

- 早期生命长期由原核型生物主导。
- 真核生物出现标志生命史进入新的组织层级。

### 主要分化

- 元古宙中，真核相关支线逐步进入较清晰记录。
- 后续进一步分出植物、动物及其他复杂真核主线。

### 鼎盛与衰退

- 真核生物不是单一短期爆发，而是持续扩张并构成复杂生命的底层框架。

## 研究意义

- 解释复杂生命为什么能够出现。
- 是理解植物、动物和多细胞化主线的共同前提页。
