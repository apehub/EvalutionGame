# 合弓类（Synapsids）

## 基本信息

- 分类层级：主干桥梁页
- 系统位置：承接 `amniotes.md`，并作为通向 `mammals.md` 的哺乳类前史桥页
- 主要时间窗口：二叠纪、三叠纪
- 页面性质：资料组织型桥页，用来解释哺乳类主线的关键上游阶段
- 是否现生：否
- 主要标签：`permian` `triassic` `mammals` `terrestrial`

## 类群概述

合弓类这页的职责，不是把所有内部支线完整铺开，而是把哺乳类前史中最不能省掉的一段讲清楚。没有这座桥，资料库里的动物主线会很容易从羊膜动物直接跳到哺乳类，让人误以为哺乳类是在更晚阶段突然完整出现的。

因此，这页最重要的作用，是用尽量通俗的方式说明：合弓类不是现代哺乳类本身，但它代表了通向哺乳类主线的一段关键上游阶段。它让“哺乳类从哪里来”不再只是一句抽象回答，而是一条真正能走通的主干路径。

## 关键特征

### 与哺乳类前史有关的方向

- 头骨、颌部和取食方式逐步表现出更接近后续哺乳类方向的变化。
- 身体结构与活动方式不再只是早期羊膜动物的简单延续，而开始呈现新的长期主线特征。

### 生态与时间位置

- 二叠纪到三叠纪是理解这条桥页最关键的时间窗口。
- 它们在古生代晚期和中生代早期的陆地生态系统中一度占据重要位置。

### 主线意义

- 这页的重点不是分类细节，而是解释为什么哺乳类前史必须有一段更早、更长的桥接过程。
- 它帮助读者理解哺乳类并不是白垩纪末或新生代才突然拥有前史。

## 演化历史

### 起点背景

- 从羊膜动物主线分出之后，合弓类逐步形成了与后续哺乳类更相关的一条方向。
- 它们的重要性不在于“是不是最终赢家”，而在于为后来的哺乳类主线提供了连续前史。

### 二叠纪窗口

- 二叠纪是这条桥页最重要的早段窗口。
- 在这一阶段，相关支线在陆地生态系统中具有很高存在感，也让哺乳类前史第一次变得清晰可追。

### 三叠纪过渡

- 到三叠纪，这条主线继续向更晚的哺乳类方向推进。
- 因而本页最合适的写法，不是细数所有内部类群，而是强调从羊膜动物到哺乳类之间并不是断裂的。

## 研究意义

- 它让 `amniotes.md -> mammals.md` 的路径不再是一跳直达。
- 它能把二叠纪和三叠纪中的动物主线读顺。
- 它也帮助资料库避免把哺乳类成功写成“只有新生代才开始”的单薄叙事。
